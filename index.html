Tutorial / วิธีใช้

1. Include please-define.min.css and please-define.min.js to your HTML

เพิ่ม please-define.min.css และ please-define.min.js ลงไปในไฟล์ HTML 

<link rel="stylesheet" href="please-define.min.css">
<script type="text/javascript" src="please-define.min.js"></script>

2. Create <select> element

สร้าง <select> ตามปกติ 

<select name="country" id="please-define">
  <option value="THAILAND">Thailand</option>
  <option value="USA">United States</option>
</select>

3. Init it

เรียกใช้งาน 

PleaseDefine(document.getElementById('please-define'));

Demo 3

Filter is the callback that will be called upon keyup event.
This case, make use of it by transform value into uppercase.

มีฟังก์ชัน Filter ที่จะถูกเรียกเมื่อเกิดอีเว้นท์ keyup
ในตัวอย่างนี้ จะประยุกต์ใช้ฟังก์ชันดังกล่าวเพื่อแปลงค่าที่กรอกมาเป็นตัวพิมพ์ใหญ่

code: 
PleaseDefine(document.getElementById('demo-3'), {
    filter: function(val){
        return val.toUpperCase();
    }
});
